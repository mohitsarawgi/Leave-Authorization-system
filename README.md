# 🗂️ Leave Authorization and Management System

A web-based system to manage employee leave applications, approvals, and role-based access control. Built using **PHP**, **MySQL**, and **Bootstrap**, this project aims to simplify the leave request process within an organization.

---

## 🛠️ Tech Stack

- 🧩 **Frontend**: HTML, CSS, Bootstrap  
- ⚙️ **Backend**: PHP (Core PHP)  
- 🗃️ **Database**: MySQL  
- 🔐 **Authentication**: Role-based access control (Admin, Manager, Employee)

---

## 💡 Key Features

- ✅ Employee registration and login
- ✅ Role-based dashboard views:
  - **Admin**: Manage users, departments, and leave types
  - **Manager**: View and approve/reject team leave requests
  - **Employee**: Apply for leave, check leave status
- ✅ Leave status notifications
- ✅ Leave history and logs for all users
- ✅ Clean UI with Bootstrap for responsiveness
